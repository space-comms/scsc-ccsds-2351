# CCSDS 235.1 Space Communications Session Control (SCSC)

This repository contains a **reference** implementation of the CCSDS 235.1 Space Communications Session Control (SCSC) protocol, written in Rust, developed as a submission for the ESA CCSDS Outreach competition for international space-communications research.

The implementation targets rigorous, standards-compliant session control for space links and is intended for use by engineers and researchers familiar with CCSDS service architectures.

## Documentation

Full technical documentation, including architecture overview, protocol state machines, and usage examples, is hosted on Read the Docs:

- https://scsc-ccsds-2351.readthedocs.io/

Please refer there for detailed design notes, API reference, and integration guidance.
